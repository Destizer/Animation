# Лабораторная работа №3

**Тема:** Работа с 2D и 3D анимацией и трансформацией  
**Обновлено:** 28.12.2025

## Описание проекта

Проект содержит реализацию 4 CSS-анимаций различной сложности с использованием современных возможностей CSS3. Каждая анимация демонстрирует различные техники создания анимированного содержимого веб-страниц.

## Цель работы

Получение навыков разработки анимированного содержимого веб-страниц. Закрепление навыков верстки и стилизации с использованием CSS3 трансформаций и переходов.

## Реализованные анимации

### Анимация 2: Свечи

**Описание:** Длинная свеча задувает короткую свечу, комната темнеет. Короткая свеча пытается зажечься снова и успешно загорается, возвращая свет.

**Этапы анимации:**
1. **Начало** - обе свечи горят ярко
2. **Задувание** - пламя длинной свечи наклоняется к короткой и задувает её
3. **Темнота** - короткая свеча гаснет, фон темнеет, появляется дым
4. **Возгорание** - короткая свеча пытается загореться (мерцание)
5. **Свет** - короткая свеча разгорается полностью, свет возвращается
6. **Цикл повторяется**

**Использованные техники:**
- **@keyframes** - множественные анимации с синхронизацией
- **radial-gradient** - создание реалистичного пламени
- **linear-gradient** - объемные свечи
- **opacity** - затухание и появление огня
- **transform: scale()** - изменение размера пламени
- **transform: translateX()** - движение пламени для задувания
- **filter: blur()** - эффект дыма
- **box-shadow и glow** - свечение от пламени

### Анимация 5: Прямоугольник с линиями

**Описание:** один прямоугольник с тремя линиями и несколькими этапами анимации.

**Этапы анимации:**
1. **Верхняя и нижняя линии перекрещиваются** - образуют крест (как в круге)
2. **Центральная линия уезжает вправо** - исчезает за границей
3. **Круг появляется и заполняется от 0 до 360 градусов** - как прогресс-бар
4. **Внутри круга появляется крестик**
5. **Фон прямоугольника меняет цвет** - с серого на розовый
6. **Все возвращается обратно** - в исходное состояние
7. **Цикл повторяется**

**Использованные техники:**
- **@keyframes** - синхронизированные анимации
- **SVG circle** - круг с stroke-dasharray для заполнения
- **stroke-dashoffset** - анимация заполнения круга от 0 до 360 градусов
- **transform: rotate()** - вращение линий для перекрещивания
- **transform: translateX()** - движение центральной линии
- **opacity** - появление/исчезновение элементов
- **animation на background** - изменение цвета фона
- **z-index** - слои элементов (линии, круг, крестик)

**Детали реализации:**
- Круг рисуется через SVG с радиусом 75px
- stroke-dasharray: 471 (длина окружности = 2 * PI * 75)
- stroke-dashoffset анимируется от 471 до 0 для эффекта заполнения
- Все анимации синхронизированы на 12 секунд

### Анимация 7: Переключатель День/Ночь

**Описание:** Плавная анимация смены дня и ночи с изменением всех элементов сцены.

**Этапы анимации:**
1. **Солнце и тучка с картинки 1 переходят влево** - начальное состояние дня
2. **Солнце меняет цвет и фактуру на луну, а тучка исчезает** - переход к ночи
3. **В освободившей правой части появляют точки-звезды** - звездное небо
4. **Меняется цвет фона и обводки** - с голубого на темно-синий
5. **По нажатию все возвращается на исходное состояние** - цикл повторяется автоматически

**Использованные техники:**
- **@keyframes** - плавные переходы между состояниями
- **animation-delay** - последовательное появление звезд
- **box-shadow** - свечение солнца и луны, кратеры на луне
- **opacity** - плавное появление/исчезновение облаков и звезд
- **transform** - движение элементов
- **inset shadow** - создание эффекта кратеров на луне

### Анимация 23: Башня с пейзажем

**Описание:** Полноценная анимированная сцена с башней, пейзажем, облаками и птицами.

**Элементы сцены:**
- **Солнце** - статичное, с эффектом свечения
- **Облака** - 3 облака движутся с разной скоростью (25s, 28s, 30s)
- **Горы** - многослойный задний план из треугольников
- **Лес** - деревья разного размера (8 деревьев)
- **Башня** - центральный элемент с окнами и развевающимся флагом
- **Птицы** - 4 птицы летят слева направо (справа налево) с волнообразным движением

**Исправления:**
- Птицы - SVG птицы в форме галочки, летят справа налево
- Флаг - стоит НА вершине крыши (top: -40px от крыши)
- Стволы деревьев - отдельные элементы, расположены ВНИЗУ

**Использованные техники:**
- **SVG** - векторные птицы
- **Правильная структура деревьев** - trunk внизу, crown сверху
- **Абсолютное позиционирование** - флаг на крыше
- **Многослойная композиция** - z-index для глубины
- **linear-gradient и radial-gradient** - для неба и элементов
- **CSS Shapes (triangles)** - для гор, крыши, деревьев
- **Бесконечные анимации** - animation: infinite
- **animation-delay** - эшелонирование элементов
- **filter: brightness()** - создание глубины

## Использованные технологии

### CSS3 возможности

**Трансформации (Transform):**
- **translateY()** - вертикальное перемещение
- **translateX()** - горизонтальное перемещение
- **scale()** - масштабирование элементов
- **rotate()** - вращение элементов
- **rotateX()** - 3D вращение по оси X

**Анимации (@keyframes):**
```css
@keyframes animationName {
    0% { /* начальное состояние */ }
    50% { /* промежуточное состояние */ }
    100% { /* конечное состояние */ }
}
```

**Переходы (Transitions):**
```css
transition: property duration timing-function delay;
```

**Другие техники:**
- **Perspective** - создание 3D пространства
- **Box-shadow** - тени и свечения
- **Gradient** - градиенты для фонов
- **Opacity** - прозрачность элементов
- **Filter** - фильтры (brightness, blur)
- **Animation-delay** - задержки анимаций
- **Z-index** - управление слоями

### HTML5
- Семантическая разметка
- SVG графика для птиц и круга

### JavaScript
- Мобильное меню
- Плавная прокрутка
- Подсветка активной страницы
- Анимация появления элементов при прокрутке

### Теоретические вопросы:

1. **Создание анимации средствами CSS**
   - Реализовано через @keyframes
   - Свойство animation и его параметры
   - Различные timing functions

2. **CSS3 Трансформация и переходы**
   - Transform (translate, rotate, scale)
   - Transitions для плавных переходов
   - 3D трансформации с perspective

### Практические задания:

1. **Поворот, искажение, перемещение и масштабирование объекта**
   - Реализовано во всех 4 анимациях
   - Особенно ярко в анимации 5 (линии и круг)

2. **Точка трансформации объекта**
   - transform-origin для вращения элементов
   - Используется для правильного вращения линий

3. **Несколько трансформаций к одному объекту**
   - Птицы: движение + волнообразность + масштаб
   - Пламя: движение + масштаб + вращение

4. **Несколько эффектов перехода к одному объекту**
   - День/ночь: цвет + позиция + форма + тень
   - Прямоугольник: цвет фона + анимация элементов

5. **Временные функции для анимации**
   - **ease** - плавное начало и конец
   - **ease-in-out** - плавное начало и конец
   - **linear** - равномерное движение
   - **ease-in** - медленное начало
   - **ease-out** - медленный конец

6. **Ход выполнения анимации**
   - Анимация 5 имеет четкие этапы с процентами
   - 0% → 25% → 50% → 75% → 100%

7. **CSS3 свойства анимации**
   - **animation-name** - имя анимации
   - **animation-duration** - длительность
   - **animation-timing-function** - функция времени
   - **animation-delay** - задержка
   - **animation-iteration-count** - количество повторений (infinite)
   - **animation-direction** - направление

8. **Формат краткой записи анимации**
   ```css
   animation: name duration timing-function delay iteration-count direction;
   /* Пример: */
   animation: birdFly 15s linear infinite;
   ```

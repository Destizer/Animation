Теоретические вопросы

1. Создание анимации средствами CSS

CSS анимации создаются с помощью @keyframes и свойства animation. Позволяют создавать плавные переходы между состояниями элементов без использования JavaScript.

2. CSS3 Трансформация и переходы

Transform позволяет вращать, масштабировать и перемещать элементы в 2D и 3D пространстве. Transitions создают плавные переходы между состояниями при изменении свойств.
